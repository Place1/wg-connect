# wg-connect

wip

## Usage

```bash
usage: wg-connect [<flags>] <config>

a cli app to connect to a wireguard VPN server using a userspace wireguard
implementation

Flags:
  --help   Show context-sensitive help (also try --help-long and --help-man).
  --debug  enable verbose logging

Args:
  <config>  a wireguard configuration file
```
